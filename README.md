practice Diwali Sales Data Analysis

Overview
This project performs an exploratory data analysis on Diwali sales data to identify key trends and insights related to customer behavior and sales patterns.

Data Source
The dataset used for this analysis is "Diwali Sales Data.csv".

Analysis
The analysis includes:

Data cleaning and preprocessing (handling missing values, changing data types).
Exploratory data analysis (EDA) to understand the distribution of data across different features like Gender, Age Group, State, Marital Status, Occupation, and Product Category.
Visualization of key metrics using bar plots to show relationships between features and sales amount/order count.
Findings
The key findings from this analysis are:

Most buyers are females with higher purchasing power.
The age group 26-35 years has the highest number of buyers and contributes the most to sales.
Uttar Pradesh, Maharashtra, and Karnataka are the top states in terms of orders and sales.
Married women have a higher purchasing power.
Buyers from IT, Healthcare, and Aviation sectors are the most frequent buyers and contribute significantly to sales.
Food, Clothing, and Electronics are the top-selling product categories.
How to Run
This analysis was performed in a Google Colab notebook. You can run it by opening the notebook in Google Colab.
